# Kapitola 8: Testování DDD

Tato ukázka demonstruje unit testování agregátů, domain events a invariantů.

## Spuštění

Otevři [http://localhost:8000/examples/testovani](http://localhost:8000/examples/testovani)

## Co se naučíš

- Unit testy agregátů bez databáze — rychlé a spolehlivé
- Testování domain events — ověření co agregát zaznamenal
- Testování invariantů — ověření že DomainException je vyhozena správně
- Jak čistá doménová vrstva bez závislostí umožňuje snadné testování
