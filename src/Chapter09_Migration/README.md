# Kapitola 9: Migrace z CRUD na DDD

Tato ukázka demonstruje porovnání CRUD a DDD přístupu a state machine guards.

## Spuštění

Otevři [http://localhost:8000/examples/migrace-z-crud](http://localhost:8000/examples/migrace-z-crud)

## Co se naučíš

- CRUD vs DDD — side-by-side porovnání stejné funkcionality
- Proč anemic model (gettery/settery) vede k nevalidním stavům
- State machine guards — DDD agregát odmítne neplatný přechod přes DomainException
- PHP enum jako náhrada za string konstanty pro stavy
